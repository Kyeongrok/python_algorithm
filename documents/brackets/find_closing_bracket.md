# 닫는 괄호 찾기
아래와 같이 괄호가 포함된 문자열이 있을 때
시작하는 괄호의 인덱스를 알려주면 끝나는 괄호의 인덱스를 리턴하는
함수를 만드세요.
```
string = [[34]HELLO[74]]
알려주는_인덱스 = 0
끝나는_인덱스 = 14
```
# 핵심 로직
* string을 하나씩 검색한다. ex) [ [ 3 4 ... 등
* 열리는 괄호가 나오면 stack에 빼둔다 [[[[]]]]
* 닫히는 괄호가 나오면 
    * stack이 0보다 크면 stack에서 pop한다.
    * 아니면 현재의 index를 return한다.
